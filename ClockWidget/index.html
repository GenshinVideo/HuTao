時計と現在の天気を表示するシンプルなウィジェットツールです。
配信画面などの空きスペースにブラウザウィンドウとして追加することを想定しています。

時計のみのウィジェットを追加する場合は、以下のURLを使用してください。

https://genshinvideo.github.io/HuTao/ClockWidget/1/

さらに天気情報を追加する場合は、以下のようにします。

https://genshinvideo.github.io/HuTao/ClockWidget/1/?location=https://forecast7.com/ja/34d69135d50/osaka/

最大3つの地域に対応しています。複数地域を指定する場合は、以下のようにしてください。

https://genshinvideo.github.io/HuTao/ClockWidget/1/?location=https://forecast7.com/ja/34d69135d50/osaka/&location2=https://forecast7.com/ja/35d69139d69/tokyo/

このクエリで使用するURLは、WeatherWidget.ioから生成できます。
「Select Location」に地域名を入力し、表示されるドロップダウンから該当地域を選択します。
その後「Language」を選択し、「GET CODE」をクリックすると必要なURLが取得できます。

背景動画はカスタムCSSを使用して非表示にできます。例えば、次のようにします。

video {
  display: none;
}

背景色の変更も同じくカスタムCSSを用いることで色の変更が可能です。

video {
  display: none;
}
.Contents {
  background: rgba(255, 255, 255, 0.4);
}
